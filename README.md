# ⚽ 银河战舰

> 抽卡 · 组队 · 开赛 · 三分钟一局，当一回足球经理人！

🎮 **[点击开始游戏](https://kun0808.github.io/galaxy-football-game/)**

---

## 🎯 这是什么游戏？

一个纯前端的足球经理人小游戏。每局从 100+ 球星中随机抽卡，拼凑你的"银河战舰"，然后一键模拟比赛看结果。没有养成、没有存档，每局都是新的开始。

---

## 🕹️ 怎么玩？

### 第一步：选阵型 📋

从三种阵型里挑一个：

| 阵型 | 风格 | 配置 |
|:---:|:---:|:---|
| 4-4-2 | 均衡 | 1门 + 4后 + 4中 + 2前 |
| 4-3-3 | 进攻 | 1门 + 4后 + 3中 + 3前 |
| 3-5-2 | 控制 | 1门 + 3后 + 5中 + 2前 |

### 第二步：按位置选人 🎴

按 **门将 → 后卫 → 中场 → 前锋** 的顺序，点击卡牌挑选球员。

> 🔄 **小贴士**：每局有一次"刷新"机会，可以换掉某个位置的全部球员。比如前锋太弱？刷新一下碰碰运气！但只能用一次哦～

### 第三步：开赛！ ⚡

点击红色大按钮，坐等 6 秒比赛动画，看结果！

赢了开心，输了也开心——毕竟对手可能叫"点球不进FC"。

## ✨ 游戏特色

- 🎴 **100+ 球员库** — 五大联赛现役球星 + 梅西、马拉多纳等退役传奇
- 🏷️ **俱乐部 & 国籍** — 每张卡牌都标注了球员的真实俱乐部
- 🤝 **默契系统** — 同联赛 ≥3 人 +3%，同国籍 ≥2 人 +2%，最高 +5%
- ⚔️ **战术选择** — 进攻 / 均衡 / 防守，影响进球数
- 🎭 **30 个足球梗对手** — "VAR受害者联""越位线舞者""集锦球王联"……
- 🔄 **每局一次刷新** — 不满意就换一批，但只有一次机会
- 📱 **手机也能玩** — 响应式设计，随时随地开一局

## 💡 小策略

1. **优先选同联赛球员** — 3 人同联赛直接 +3%，比单堆高评分更划算
2. **传奇球员很香** — 梅西 98 分、马拉多纳 97 分，但抽不抽到看脸
3. **刷新用在刀刃上** — 如果前锋池只有 85 分以下的，果断刷新
4. **战术影响进球数** — 进攻战术进球多但失球也多，防守反之
5. **别小看随机性** — 评分差不到 5 就是五五开，冷门随时发生
