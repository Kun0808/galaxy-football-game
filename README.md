# ⚽ 银河战舰

> 抽卡 · 组队 · 开赛 · 三分钟一局，当一回足球经理人！

🎮 **[点击开始游戏](https://kun0808.github.io/galaxy-football-game/)**

## 🎯 这是什么游戏？

一个纯前端的足球经理人小游戏。每局从 100+ 球星中随机抽卡，拼凑你的"银河战舰"，然后一键模拟比赛看结果。没有养成、没有存档，每局都是新的开始。

---

## 🕹️ 怎么玩？

### 第一步：选阵型 📋

从七种阵型里挑一个：

| 阵型 | 风格 | 配置 |
|:---:|:---:|:---|
| 4-4-2 | 均衡 | 1门 + 4后 + 4中 + 2前 |
| 4-3-3 | 进攻 | 1门 + 4后 + 3中 + 3前 |
| 3-5-2 | 中场 | 1门 + 3后 + 5中 + 2前 |
| 4-5-1 | 控球 | 1门 + 4后 + 5中 + 1前 |
| 3-4-3 | 全攻 | 1门 + 3后 + 4中 + 3前 |
| 5-3-2 | 防守 | 1门 + 5后 + 3中 + 2前 |
| 5-4-1 | 铁桶 | 1门 + 5后 + 4中 + 1前 |

### 第二步：按位置选人 🎴

按 **门将 → 后卫 → 中场 → 前锋** 的顺序，点击卡牌挑选球员。

> 🔄 **小贴士**：每局有一次"刷新"机会，可以换掉某个位置的全部球员。比如前锋太弱？刷新一下碰碰运气！但只能用一次哦～

### 第三步：开赛！ ⚡

点击红色大按钮，坐等 6 秒比赛动画，看结果！

赢了开心，输了也开心——毕竟对手可能叫"点球不进FC"。

## ✨ 游戏特色

- 🎴 **100+ 球员库** — 五大联赛现役球星 + 退役传奇
- 🏷️ **俱乐部 & 国籍** — 每张卡牌都标注了球员的真实俱乐部
- 🤝 **默契系统** — 同联赛 ≥3 人 +3%，同国籍 ≥2 人 +2%，最高 +5%
- ⚔️ **战术选择** — 进攻 / 均衡 / 防守，影响进球数
- 🎭 **30 个足球梗对手** — "VAR受害者联""越位线舞者""集锦球王联"……
- 🔄 **每局一次刷新** — 不满意就换一批，但只有一次机会
- 🎲 **分层抽卡系统** — 传奇和精英球员稀有出现，凑出强队更有成就感
- 📱 **手机也能玩** — 响应式设计，随时随地开一局

## 🎲 抽卡概率说明

不是纯随机！抽卡采用分层系统，控制稀有球员的出现频率：

| 球员层级 | 定义 | 出现概率 |
|:---:|:---|:---:|
| 传奇 | 退役名宿（梅西、马拉多纳等） | 每个位置 ~40% 概率出 1 张 |
| 精英 | 现役 90+ 评分球星 | 每个位置 ~35% 概率出 1 张 |
| 普通 | 其余现役球员 | 填满剩余名额 |

**实测数据**（200 次抽卡统计）：

| 指标 | 数值 |
|:---|:---:|
| 传奇出现率 | ~9% |
| 精英出现率 | ~6% |
| 每局平均传奇数 | 1.6 张 |
| 卡牌平均评分 | 86.5 |

> 想凑齐 5 个传奇？可能需要多刷几局。这就是"银河战舰"的含金量。

## 💡 小策略

1. **优先选同联赛球员** — 3 人同联赛直接 +3%，比单堆高评分更划算
2. **传奇球员很珍贵** — 平均每局只有 1-2 个传奇，抽到就好好用
3. **刷新用在刀刃上** — 如果前锋池全是 84 分以下的，果断刷新
4. **战术影响进球数** — 进攻战术进球多但失球也多，防守反之
5. **别小看随机性** — 评分差不到 5 就是五五开，冷门随时发生
6. **阵型决定打法** — 5-4-1 铁桶阵防守稳但进攻弱，3-4-3 全攻进球多但容易丢
